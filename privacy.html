Amicus — политика конфиденциальности

Amicus — приложение-соцсеть, в котором лента пользователя показывает посты только тех, с кем есть подтверждённое знакомство. Это пилотный проект, находящийся на стадии закрытого бета-тестирования.

Какие данные собираются

  • Email и пароль — для регистрации и входа (пароль хранится в хешированном виде, приложение его не видит).
  • Имя — указывается при регистрации, видно другим пользователям приложения.
  • Фото профиля (аватар) — по желанию.
  • Текст и фото постов, комментариев — то, что вы сами публикуете.
  • Лайки — какие посты вы отметили.

Как данные используются

Данные используются исключительно для работы приложения: показать вашу ленту, ваш профиль и профили знакомых, ваши посты — тем, с кем у вас подтверждено знакомство через invite-ссылку. Данные не передаются третьим лицам, не продаются, не используются для рекламы (рекламы в приложении нет).

Где хранятся данные

Данные хранятся у облачного провайдера Supabase (PostgreSQL + Storage), с ограничением доступа на уровне базы данных (Row Level Security) — то есть пост виден в базе только тем, кто имеет право его увидеть, даже без участия кода приложения.
